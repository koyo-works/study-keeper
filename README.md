## サービス概要 
「study-keeper」は、勉強しようと思ってもつい他のことをしてしまう人のための行動選択支援アプリです。
無意識の行動を意識的な選択に変えることで、理想の学習習慣の定着をサポートします。
選択データの蓄積・可視化により、理想の学習習慣の定着をサポートします。

## このサービスへの思い・作りたい理由
自分自身、勉強しようと思ってもゲームや動画、SNSなどに気を取られてしまい、気づけば1日が終わっていたという経験が何度もありました。
「やる気がないわけじゃないのに続かない」——その原因の多くは、無意識に行っている行動の積み重ねにあると感じています。

そこで、行動を“選ぶ”という小さな意識の変化から、自分の習慣を整えていけるようなアプリを作りたいと思いました。
このアプリを通じて、自分と同じように「続けたいのに続かない」人が少しでも前に進めるきっかけを作れたらと思っています。

## ユーザー層について
### メインターゲット
#### Webエンジニアを目指して学習中のスクール生・独学者

##### 対象ユーザー1：Webエンジニアを目指して学習中のスクール生・独学者
 - 「やる気はあるのに、継続が難しい」という共通の悩みを抱えている。
 - 学習内容が多く、自由度が高いため、自己管理能力が求められる。
 - 自分自身もこの層に属しており、仕事終わりに学習のやる気が湧かない時が続く

##### 対象ユーザー2：学習・資格勉強をしている社会人・学生
 - 仕事や学校の合間に勉強時間を確保する必要があり、誘惑や疲労で勉強が後回しになりやすい。
 - 行動を「選ぶ」という軽い操作で学習意識を切り替えられる設計が、心理的ハードルを下げる。
 - SNS・動画などに気を取られる現代の生活習慣にマッチしている。

## サービス利用イメージ
 - ユーザーはアプリを開くと、「今何をしようとしている？」という質問が表示される。
   例：「勉強する」「SNSを見る」「動画を見る」などの選択肢。

- その時の行動をボタンで選ぶだけで記録され、日や週ごとの行動傾向を自動で可視化。
  「今週は“勉強する”を選んだ回数が先週より20％増えた」など、自分の行動変化を客観的に把握できる。

- これにより、ユーザーは“無意識にサボってしまう”時間を減らし、“意識して勉強に向かう”習慣を身につけられる。
  結果として、勉強の継続率と自己管理力の向上という価値を得られる
